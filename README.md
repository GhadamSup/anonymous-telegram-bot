# Anonymous Telegram Bot

A Telegram bot that allows anonymous messaging between users with deep linking support.

## Features

- 🔗 Generate unique anonymous messaging links
- 💬 One-time anonymous messaging
- ↩️ Reply button system for conversations
- 🖼 Media support (photos, videos, voice notes, stickers)
- 📊 Admin panel with statistics
- 🚫 Ban/unban system
- 📢 Broadcast messages
- 📁 Advanced message logging
